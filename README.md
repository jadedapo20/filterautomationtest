# FILTERAUTOMATIONTEST

Automated end-to-end testing framework for JD Wetherspoon using Cucumber, Playwright with JS, and Node.js.  
Supports **cross-browser testing** on Chromium, Firefox, and WebKit.

## Installation
Install all dependencies:

npm install

## Run All Browsers in Parallel
This command will run tests simultaneously on Chromium, Firefox, and WebKit:

npm run test:all

## Run a Single Browser
You can also run tests on a specific browser:

# Run on Chromium
npm run test:chromium

# Run on Firefox
npm run test:firefox

# Run on WebKit
npm run test:webkit
